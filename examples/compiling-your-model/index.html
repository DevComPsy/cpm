Create your model

Import the necessary libraries

import cpm

Create your model

model = cpm.Model()

Add pre-programmed components to the model

model.add_component(cpm.Component1())
model.add_component(cpm.Component2())
model.add_component(cpm.Component3())

Compile the model

model.compile()

Run the compiled model

model.run()
